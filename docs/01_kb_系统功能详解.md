# 通用Word智能填充Agent - 系统功能详解

创建日期：2026-05-14  
最后更新：2026-05-14

---

## 1. 功能总览

系统已实现以下核心能力：

1. 模板管理（管理员）
2. 文档生成（用户）
3. AI 对话填充（用户）
4. 导入填充（用户）
5. 复杂模板能力（内容控件/图片/兼容性）
6. AI 文件智能提取（多引擎兜底 + 置信度）

---

## 2. 核心功能模块

### 2.1 模板管理

- 上传 Word 模板
- 自动识别字段（普通字段 + 表格字段）
- 字段配置、引导话术配置
- 模板启用/禁用

### 2.2 文档生成核心

- 普通字段替换（`{字段名}`）
- 表格字段动态填充（`{表格名.字段名}`）
- 数据校验（phone/email/date/number/必填）
- 生成与下载 Word 文档

### 2.3 AI 智能对话填充

- 自然语言对话收集字段
- AI 字段提取（单轮/多轮）
- 上下文感知引导提问
- SSE 流式输出
- 会话持久化与断点续聊

常用快捷指令示例：

| 用户输入示例 | AI 理解意图 | 功能 |
|------|------|------|
| 我要一次性填完 / 直接列出所有字段 | 批量填写模式 | AI 列出字段并引导批量输入 |
| 给我模板 / 下载模板 | 下载模板 | 返回模板下载链接 |
| 跳过废话 / 直接开始 | 快速模式 | 跳过欢迎语并开始收集字段 |
| 修改项目名称 | 字段修改 | 定位并更新指定字段 |
| 负责人是张三，预算50万 | 批量提取 | 一次提取多个字段值 |

### 2.4 导入填充

- 支持 Excel/JSON/Word 导入
- 字段智能匹配（精确/模糊/语义）
- 匹配置信度评分
- 可视化人工调整
- 补全缺失字段并生成文档

### 2.5 复杂模板能力

- 模板预检（格式与结构）
- 占位符规范化（全角/空格容错）
- 内容控件处理（文本、富文本、下拉、复选框、日期、图片）
- 旧版表单域兼容
- 图片替换（Base64/本地文件）

### 2.6 AI 文件智能提取

- 上传 PDF/Word/Excel/图片等参考文件
- AI 批量提取字段
- 多引擎降级：Copilot SDK -> Azure OpenAI -> OpenAI -> 规则提取
- 置信度分级 + 人工确认
- 一键填充并生成文档

置信度评分机制：

| 置信度等级 | 分数范围 | 处理方式 |
|------------|----------|----------|
| 高置信度 | 90-100 | 自动填充，无需确认 |
| 中置信度 | 70-89 | 自动填充并标记提示 |
| 低置信度 | 50-69 | 需人工确认 |
| 无法识别 | <50 | 标记未匹配，需人工输入 |

## 3. 技术与实现要点

### 3.1 技术栈

- 后端：C# + .NET 8 + Microsoft Agent Framework + GitHub Copilot SDK
- 前端：Vue3 + TypeScript + Element Plus
- 文档处理：OpenXML SDK（主）/ DocX / NPOI
- 数据存储：SQLite + 本地文件系统

### 3.2 AI 与引擎策略

- 主链路：CopilotClient + CopilotSession
- 编排层：MAF（按迭代逐步增强）
- 兜底链路：Copilot SDK -> Azure OpenAI -> OpenAI -> 规则匹配

### 3.3 文档与中文兼容性

| 问题 | 处理方式 |
|------|----------|
| 字体显示异常 | 模板预设中文字体（微软雅黑/宋体） |
| 表格列宽变形 | 使用固定列宽 |
| 文本换行异常 | 启用东亚换行规则兼容设置 |

---

## 4. MVP 范围

已纳入 MVP：

- 单文档生成（最多5条）
- AI 对话填充
- 外部数据导入（Excel/JSON/Word）
- 表格字段填充
- 模板下载
- AI 文件智能提取
- 多引擎 LLM 兜底
- SQLite 数据库存储

未纳入 MVP：

- 批量处理（>5条）
- 运营统计
